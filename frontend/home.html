Chrome Symbolizer

Symbolize the value of a Breakpad upload "product data" key. These appear on the crash server in the blue metadata at the top of the report. Example: zombie_dealloc_bt.

Symbolize an Apple crash or hang report from ~/Library/Diagnostic Reports, or the output from running the Sample command in Activity Monitor or at the command line.

Symbolize the machine-format output of Breakpad's minidump_stackwalk -m command.

Look up the Breakpad module names and identifiers for a product and version. Product names are the crash reporting ones, e.g. Chrome_Mac.

Symbolize arbitrary addresses of a stack fragment. It is often useful to get the Module Name and Module Identifier from the "Look Up Module Info" option.

Symbolize the crash from an android log. The log is obtained with the command adb logcat.
